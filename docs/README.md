## 🔧 技术栈

- **前端框架**: React 18.3 + TypeScript 5.8
- **构建工具**: Vite 7.0
- **UI 框架**: Ant Design 5.25
- **状态管理**: Zustand + React Query
- **样式方案**: TailwindCSS 4.1
- **路由**: React Router 7.6
