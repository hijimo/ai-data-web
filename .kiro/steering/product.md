# 产品概述

## 项目简介

聚协云 - 一个基于 React 的企业级 SaaS 管理平台，提供多租户管理、用户管理、AI 聊天会话和模型配置等核心功能。

## 核心功能模块

- **首页仪表盘**: 数据概览和快速访问入口
- **聊天会话**: AI 对话交互功能，支持流式响应
- **租户管理**: 多租户架构支持，租户信息管理
- **用户管理**: 用户账号管理、权限控制
- **模型设置**: AI 模型配置和参数调整

## 技术定位

这是一个现代化的前端单页应用（SPA），采用 Vite 构建工具替代传统的 Create React App，提供更快的开发体验和构建性能。项目遵循企业级代码规范，强调类型安全、可维护性和用户体验。

## 用户界面

- 采用 Ant Design 5 设计系统
- 响应式布局，支持桌面端和移动端
- 中文界面，面向中国市场
- 主题色可配置（通过环境变量 VITE_THEME_COLOR）

## 架构特点

- 前后端分离架构
- RESTful API 通信
- 基于 OpenAPI 规范的 API 代码生成
- 支持开发环境 API 代理（/api/ -> localhost:8080）
